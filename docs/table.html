pds-table — states & selection

The DS themed Tabulator wrapper. RTL, virtual scroll, no pagination. Use the buttons to drive the four contract states; select rows to see the "N נבחרו" bulk bar with a clear control and one demo action.

Inline edit (PLT-081): click a value in the אחראי (owner) column to edit it in place — ✓ confirm / ✕ cancel, Enter/Esc, with a busy state while saving. This demo is wired to a fake async saver so you can see every path: type a value containing fail → the save is rejected (retry in place); type a value containing conflict → a 409-shaped rejection refreshes the cell to the server value with an inline notice (no silent overwrite); any other value saves after a short delay.
