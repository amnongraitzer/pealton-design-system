Phase 2 (PLT-056). Rubik (SIL OFL, variable weight 300–900, Hebrew + Latin subsets, ~44 KB) now ships inside the package as static/fonts/*.woff2 and loads via static/fonts.css — no CDN, offline-safe. This removes the silent Segoe-UI / system fallback both apps used to render. The banner above verifies Rubik actually rendered Hebrew. Components, macros, and the app rollout follow in PLT-057 … PLT-069.
